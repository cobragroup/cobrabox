# Documentation

Suggested order for new contributors:

## Before the outing
1. First follow `setup_repo.md`. To set up the repository locally
2. If you have never made a pull request before. Follow `your_first_pull_request.md`.
3. If you need to set up your ssh on github (the most awesome way of using github) `setup_github_ssh_key.md`

## During the outing
### If feature dev
Follow `contributing_feature.md` for every feature
For a quick reference on how to do a pr: `how_to_make_a_pr.md`
